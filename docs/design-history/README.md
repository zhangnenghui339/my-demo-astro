# Design history

built site (Astro only compiles `src/pages/`), so nothing here is ever
deployed or linked from a live page.

- `careerradar_home_v1.html` — earlier, simplified homepage concept (2 pillars,
  extra "why trust us" section).
- `careerradar_home_preview.html` — the approved final homepage design,
  implemented in `src/pages/index.astro` + `src/layouts/Layout.astro`.
